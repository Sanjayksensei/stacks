Stack — LIFO (Last-In, First-Out) — like a pile of plates. You can only add or remove from the top. The last element added is the first removed. Three Ways to Create a Stack in Python

Each method has trade-offs. Use list for LeetCode, deque for production, class for interviews.
Method 1 — List
stack = []
stack.append(x)
stack.pop()
stack[-1]
✓ Simplest. Amortized O(1).
Method 2 — Deque
from collections import deque
stack = deque()
stack.append(x)
stack.pop()
✓ True O(1). Thread-safe.
Method 3 — Class
class Stack:
  push(x), pop()
  peek(), is_empty()
  size()
✓ Encapsulated. Interview-ready.
